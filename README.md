## Hugo IRIS Theme

This repository includes the [Hugo] theme.

- Writing slides using Markdown. ([reveal.js])
- [MathJax] on slides.
- Diagram and flowchart on slides. ([mermaid])

This theme can manage and publish slides using Hugo.
